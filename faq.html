Frequently Asked Questions
Why is the server offline? - The server restarts every day at 15:00 (CET). About a quarter later, at 15:15 (CET) you will be able to join again. - The server shuts down automatically when it's 00:00 (CET). I try to turn it on as much as possible, likely from 7-9 in the morning.
Why can't I join the server? The server has a whitelist. You are probably not whitelisted. To get on the whitelist, join the Discord server and ask for permission. You can find a link to the Discord server at the bottom-right corner of the Server-page, or join using this link.
How do I join the old, 1.12.2 server? All servers are reachable from the same IP-address: play.wouterk12.com To join the old, 1.12.2 server, use Minecraft version 1.12.2 with that IP-address. To join the new, x.xx.x servers, use the latest Minecraft version with that IP-address. Use in-game commands to navigate between these new servers (/server <tab>).
