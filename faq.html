Frequently Asked Questions

Why is the server offline? - The server restarts every day at 15:00 (CET). About a quarter later, at 15:15 you will be able to join again. - The server shuts down automatically when it's 00:00. It turns on at 7 in the morning.
Why can't I join the server? The server has a whitelist. You are probably not whitelisted. To get on the whitelist, join the Discord server and ask for permission. You can find a link to the Discord server at the bottom-right corner of the Server-page, or join using this link.
How do I join older servers? All servers are reachable from the same IP-address: play.wouterk12.com
Example: To join the old, 1.12.2 server, use Minecraft version 1.12.2 with that IP-address. To join the new, x.xx.x servers, use the latest Minecraft version with that IP-address. Use in-game commands to navigate between servers with the same version (/server <tab>).
